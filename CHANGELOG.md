# Changelog

All notable changes to the JC-120 Jazz Chorus Plugin will be documented here.

## [1.0.0] — 2026-04-24

### Initial Release
- Solid-state preamp with transistor soft-clipping
- 3-band tone stack (Bass, Mid, Treble) with Bright switch
- CE-1 style BBD stereo chorus (7ms centre delay, dual LFO)
- Schroeder spring reverb emulation
- Realistic JC-120 front panel UI
- VST3 + AU support (macOS)
- 10 fully automatable parameters
